A minimalist linux rice setup for the i3 window manager.

Components:

- Window Manager: i3
- App Launcher: rofi
- Terminal Emulator: Terminator
- Status Bar: polybar
- Font: DepartureMono Nerd Font
- Autotiling: nwg-piotr/autotiling
- Neovim: NvChad
- Compositor: ibhagwan/picom
